======

Dconfig is configuration by console for the Django web
development framework.

Support Django >= 2.0


Installation
------------
1. Python package

        pip install dconfig

2. Add 'dconfig' to INSTALLED_APPS:

        'dconfig',


Usage
--------

`python manage.py conf`


Help
--------

`python manage.py conf`
>>> help


Wizard Setup
--------

`python manage.py conf`
>>> init

** WARNING: this action will wipeout all data in settings.py**
